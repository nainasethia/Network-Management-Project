# Network-Management-Project
This is my fifth Git Repository. In this, you'll see my Network Management Project. 
-	Configured web, DNS, mail, and file servers on Red Hat Enterprise Linux to simulate a complete enterprise network setup.
-	Established internal communication using private IP addressing and implemented POP3 email and DHCP services for client systems.
-	Set up domain services and created subdomains such as sales.company.in, research.company.in, and accounts.company.in.
-	Designed and maintained secure departmental access controls and hierarchical file permissions for data and driver directories.
-	Acted as Chairperson of the simulated company, overseeing network resource allocation, user privileges, and server maintenance.
-	Gained hands-on experience in network configuration, Linux administration, and enterprise-level IT infrastructure management.
